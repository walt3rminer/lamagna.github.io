Sort bibliography in a latex thesis or paper
Script to sort automatically bibliography in a thesis.  Use:
  • make a backup of your work!
  • put in a file, correctly sorted, all parts from the thesis. Get the sample file
  • execute the script this way: sortbib.pl lista.txt bibliografia.tex > bib2.tex
  • Now check bib2.tex, add any missing lines in the top and bottom of the file
  • Use the bib2.tex file.

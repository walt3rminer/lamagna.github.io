Ordenar bibliografia en tesis o paper hechos en latex
Script para ordenar automaticamente bibliografia en una tesis.  Uso:
  • haz un backup de tu trabajo!
  • Pon dentro de un archivo, correctmente ordendado, todas las partes de la tesis. Ejemplo de archivo
  • ejecuta el script de esta manera: sortbib.pl lista.txt bibliografia.tex > bib2.tex
  • Ahora revisa bib2.tex, agrega lineas que puedan faltar arriba y debajo del archivo
  • Utiliza el archivo bib2.tex.

Installing Asterisk PBX on Redhat Centos 5

This was easy and fast:
# mkdir asterisk
# cd asterisk
# wget http://redhat.theplanet.com/asterisk/asterisk-1.4.10.1.tar.gz
# tar xzvf asterisk-1.4.10.1.tar.gz
# cd asterisk-1.4.10.1/
#./configure
# make
# make install
# make samples

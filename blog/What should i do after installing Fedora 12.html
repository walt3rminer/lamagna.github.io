What should i do after installing Fedora 12

I installed Fedora 12 in a ThinkPad T61.  Honestly i am surprised with the performance because Redhat like operating systems work smootly on a Thinkpad, wifi, bluethooth, 3G, everything, even Lotus Notes.

What i would like to tell is:  If you are planing to use IBM software on the notebook, take care with the upgrades, i recommend to avoid any kind of upgrade suggested by the operating system, may be apply the security upgrades but at the minimum, for example, only the services that you use... because it brought problems to my lotus notes 8.5.1 (beta) related with libraries... if you choose for the IBM world, then live only in that world, in contrast, you can upgrade.....

Now, after installing Fedora 12:
Update yum 
sudo yum install yum-plugin-fastestmirror
sudo yum install yum-presto

Add 3rd party repositories: 
rpm -ivh http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-stable.noarch.rpm
rpm -ivh http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-stable.noarch.rpm
rpm -ivh http://rpm.livna.org/livna-release.rpm
MP3 support: 
yum install ffmpeg*
yum install xmms xmms-mp3 xmms-faad2 xmms-pulse xmms-skins
yum install audacious audacious-plugins-freeworld*
yum install rhythmbox gstreamer-plugins-ugly gstreamer-plugins-bad gstreamer-ffmpeg
yum install amarok xine-lib-extras-freeworld
